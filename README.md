## Requirements

- **Compiler:** GCC (GNU Compiler Collection) or any C99-compatible compiler
- **OS:** Linux, macOS, or Windows (via MinGW / WSL)
- **Standard:** C99

---

## How to Compile & Run

Clone the repository:

```bash
git clone https://github.com/GaelMHub/C-First-Semester.git
cd C-First-Semester
```

Compile any program:

```bash
gcc -Wall -std=c99 -o output_name filename.c
./output_name
```

Example:

```bash
gcc -Wall -std=c99 -o calculator calculator.c
./calculator
```
